## Description

Configures replicas for your service.

## Specification

List of all configuration options for a `Scaler` trait.

```yaml
...
    scaler:
      replicas: 100
```

## Properties

Name | Description | Type | Required | Default 
------------ | ------------- | ------------- | ------------- | ------------- 
 replicas | Replicas of the workload | int | true | 1
